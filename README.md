# travel-agency
### mc-access db, using SQL queries to withdraw information

You can find all steps from designing db using ER diagram to creating forms, reports, etc in the *report* file.

* Client, visa and passport, order registration
* Clients information(Full name, passport/visa details, phone number, etc..)
* Travelling packages details (Departure date, return date, price, hotel information, excursions, etc..)
* Travelling packages assistance (Finding packages according to price, country, vehicle)
* Reports (Monthly report, Travelling package report with all dates, hotel/excursions details)
